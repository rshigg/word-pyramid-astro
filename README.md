# Word Pyramid

[![Screenshot of Word Pyramid](./word-pyramid.png)](https://wordpyramid.vercel.app)

## Developing

Install dependencies with `pnpm install` then start a development server:

```bash
pnpm run dev

# or start the server and open the app in a new browser tab
pnpm run dev --open
```

## Building

```bash
pnpm run build
```

Then you can preview the build:

```bash
pnpm run start
```
